EasynuX

EasynuX é uma ferramneta de terminal Linux voltada a usuários iniciantes.

Seu foco é facilitar o uso do sistema operacional com a tradução da complexidade do uso do terminal em uma linguagem simples, provendo mais acessibilidade e controle entre o usuário e o sistema operacional.

Como obter?

Primeiramnete cerfifique-se de estar usando um sistema operacional Linux base Debian como os listados abaixo. Caso não tenha posse de nenhum deles, obtenha o de sua escolha clicando em uma das opções abaixo:

  • Obter Linux Mint
  • Obter Ubuntu
